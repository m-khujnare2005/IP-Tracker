# IP-Tracker (GUI using python)

# Description

This project is a beginner-friendly IP Tracker application built using Python. It allows users to enter an IP address and an API key to fetch real-time details such as city, country, ISP, and geographic coordinates.

The application provides a simple and interactive GUI using Tkinter and also allows users to open the location directly in Google Maps for better visualization.

# Features

🔑 User input for API Key

🌐 Track any IP address

🖥️ Simple and interactive GUI

📍 Displays city, country, ISP, and coordinates

🗺️ Open location in Google Maps

❌ Error handling for invalid input
# Technologies Used

Python

Tkinter (GUI)

Requests (API handling)
